Place your Amiga Kickstart ROM file here.

Required filename format: any name, extension must be .rom
Example: kick13.rom, kick31.rom, amiga-os-130.rom

RetroScreen will use the first .rom file it finds in this folder.

Recommended ROMs:
  Kickstart 1.3 (A500)  -- best compatibility with classic games
  Kickstart 3.1 (A1200) -- required for some later titles

How to obtain a Kickstart ROM legally:
  Purchase "Amiga Forever" from Cloanto: https://www.amigaforever.com
  This product includes licensed Kickstart ROM images for personal use.
